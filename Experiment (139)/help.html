Position of an Object wih Mirror experiment help

About the experiment

The experiment shows an Object Placed on a Chessboard contains Mountains , Mirror around it.

Animation control

You can control the Object by keyboard.

Forward : W

Backward : S

Left : A

Right : F

The setup stage

The image of the Object will be shown on that Plane Mirror

The image of the beautifull mountains also shown in the mirror due to law of reflection

The object obeys the laws of physics

You can move the object forward,backward,left and right using the keybord

Distance of the object from the mirror is shown at top left corner of the experiment in centimeters
